Página de Produtos - Kelltech Informática
Este é um projeto de uma página de produtos que eu desenvolvi usando React. A ideia principal foi criar a interface de uma vitrine de e-commerce focada em artigos de tecnologia, aplicando conceitos de componentização, reutilização de código e estilização dinâmica.

Na página principal, eu renderizo os produtos de forma dinâmica a partir de um array de dados, utilizando JavaScript moderno. Para a organização visual, eu construí um layout responsivo usando CSS Flexbox, garantindo que os cards dos produtos fiquem alinhados lado a lado em uma grade que se adapta ao tamanho da tela. Toda a estrutura do projeto foi dividida de forma lógica, contando com componentes como a página principal, um cabeçalho, o card de produto e o botão.
